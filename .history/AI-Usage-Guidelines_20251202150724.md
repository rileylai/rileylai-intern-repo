### **2. What are the benefits and risks of using AI in a professional setting?**

**Benefits:**
- Saves time on repetitive tasks  
- Helps understand new concepts  
- Improves writing and documentation  
- Gives quick examples or starter code  

**Risks:**
- AI can generate incorrect information  
- Sensitive data may leak if entered into AI tools  
- Over-reliance can reduce independent thinking  
- Code might not follow company guidelines  

This is why AI must be used carefully and responsibly.

---

### **3. What types of information should never be entered into AI tools?**

You should never enter:
- Passwords or access tokens  
- API keys  
- Company internal documents  
- Confidential business information  
- Client or user data  
- Personal information (yours or others‚Äô)  

AI tools are not always private, so sensitive data must be protected.

### **4. How can you fact-check and validate AI-generated content to ensure accuracy?**

You can:
- Read the output carefully  
- Compare it with official documentation  
- Test AI-generated code before using it  
- Ask team members to review important content  
- Check multiple sources online  

Critical thinking must always be used when working with AI.
